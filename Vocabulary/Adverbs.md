# Adverbs

## Miscellaneous adverbs
| Spelling | Definition |
| -------- | ---------- |
| nen | no |
| con | soon |
| huba | overly |
| sun | why |
| saci | how |

## Temporal adverbs
(-hi is the termination for a temporal adverb)
| Spelling | Definition |
| -------- | ---------- |
| tahi | now |
| kahi | then |
| sohi | when |

## Demonstrative adverbs
(-wa is the termination for a demonstrative adverb)
| Spelling | Definition |
| -------- | ---------- |
| tawa | here |
| kawa | there |
| sowa | where |
